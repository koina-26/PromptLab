# 🚀 PromptLab – GPT-2 Text Generation Project

## 📌 Overview
PromptLab is a simple Natural Language Processing (NLP) project using **GPT-2 (Hugging Face Transformers)**.  
It generates human-like text from prompts and analyzes how different settings affect creativity and output quality.

## 🎯 Goal
To study how GPT-2 generates text and how randomness (sampling) affects creativity and meaning.

## ✨ Features
- 🤖 AI text generation using GPT-2  
- 🔁 Multiple outputs for each prompt  
- 🧠 Creativity score (text diversity check)  
- ⚖️ Comparison: Pipeline vs Manual generation  
- 📊 CSV dataset creation  
- 📄 Automatic PDF report generation  

## 🛠️ Tools Used
- Python 🐍  
- Hugging Face Transformers 🤗  
- PyTorch 🔥  
- Pandas 📊  
- ReportLab 📄
